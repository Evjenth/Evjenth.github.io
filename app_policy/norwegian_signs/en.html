This is a privacy policy for the application with package name “com.evjenth.anstr.traficsigns”. The name of the application may vary for different languages, and may change in future upgrades, however, the package name will remain constant.

User information

The application does not collect any personal identifiable information about the user. Some information are collected for statistical purposes automatically through Google Play’s own tools (Firebase), and are presented for the developer of the application. Information that is collected may include age, country and more, but it is not possible to identify specific users through this information, as only the accumulated data from multiple users are displayed to the developer.

If the user wishes to know the sources for many of the questions and images send a mail to the developer for more information.

Either grammatical errors, typos, outdated information or misinterpreted information can be displayed to the user. The user is encouraged to contact the developer if they think any information is incorrect, so it can be fixed.
